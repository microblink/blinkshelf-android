# BlinkShelf


The BlinkShelf SDK lets you build a fantastic in-store experience in your Android app. Effortlessly scan the products in the grocery store and get all the needed data about them!

## Requirements

- AndroidX
- Min SDK 21+
- Compile SDK: 33+
- Java 8+
